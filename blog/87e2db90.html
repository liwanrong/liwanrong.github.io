Task02：Ubuntu基本设置
发表于 2020-04-22| 更新于 2020-04-22|创客与学科教学应用
字数总计:459|阅读时长: 1 分钟
| 阅读量:|评论数:

拓展：虚拟机镜像分享站点：https://www/osboxes.org/ubuntusf.net

操作系统

  • 用户、应用软件、操作系统（内核（管理硬件系统）-系统调用，终端命令，图形界面））、硬件系统
  • 类型
    • 桌面：Windows Mac Linux
    • 服务器：Unix（最早 美国） Linux Windows Server
    • 嵌入式：Android iOS HarmonyOS

Linux

  • 内核：运行程序和管理资源的核心程序，提供了硬件与应用程序间的抽象层。内核具备基本的硬件调用功能（终 端命令→系统调用→硬件资源）
  • 发行版：内核-基础应用，Ubuntu、CentOS、 Deepin、 UOS…..

Ubuntu

Ubuntu版本

  • Desktop：桌面版本，自带Gnome图形环境
  • Server：服务器版本，默认没有图形界面
  • IoT：loT= Internet of Things，物联网。支持树莓派、Intel NUC等
  • Cloud：VPS版， 针对Amazon AWS、Microsoft Azure等优化

主要衍生版本

  • Kubuntu: KDE代 替Gnome
  • Xubuntu : Xfce代 替Gnome
  • Lubuntu: Lxde代替Gnome
  • Ubuntu Studio:集成开源媒体创作工具
  • UbuntuKylin （优麒麟） :中国专用版

版本号

  • 类型：普通版本，LTS版本（Long Term Support，2年）
  • 版本号：年代+月份 目前为止每年4、10发布新版

图形化界面

  • 统一菜单
  • Dock
  • 程序启动器

熟悉终端

  • 启动：Terminal、收藏到Dock栏;
  • 字符尺寸
    • 放大: Ctrl-Shift+=; 缩小: Ctrl–
    • 属性/快捷键

安装新软件

  • 比AppStore更早的应用商城
  • “Ubuntu软件”，搜索并安装
  • 本课程中不推荐乱安装

配置更新源

  • 软件更新
  • 选择中国源
  • 也可测试

用户账户

  • 默认账号均为标准账号
  • 安装、配置时均提示输入密码临时调用管理员权限
  • 命令行中如何临时使用管理员权限- sudo
  • 密码输入无提示

命令

  • 安装软件：sudo apt-get install package_name
  • 删除：sudo apt-get remove package_name
  • 更新：sudo apt-get update package_name
